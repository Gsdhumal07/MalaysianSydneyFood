# Malaysian Syndey Food App
![](screenshots/malaysian_sydney_food_app.gif)

## Requirements
- Android Studio Dolphin or later

## Tech Stack
- Android View
- MVVM Achitecture
- RecycleView
- Room Database / SQLite
- Material Card View
- Glide
- Shared Preferences
- Coroutine & Live Data
